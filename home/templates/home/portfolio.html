Portfolio

As a student at Code Institute, I submitted five coding projects. The projects ranged from creating a website with HTML and CSS to using more complex languages and tools like Django and Bootstrap.

HTML/CSS

Yoga at home is a site created to teach yoga and to make yoga more accessible. The site targets people of all levels from beginner to advanced by having three levels with varying course material.
Yoga at home allows its users to view a live sessions calendar, to buy courses and to sign up for newsletters.

Here is the live version of my project:
www.aria-vero-s.github.io/yogaathome

Python

Enviro is a Python terminal survey, which runs the Code Institute mock terminal on Heroku.
The user answers four questions on the topic of climate change. After completion, the answers are automatically compiled in a separate google sheet document.

Here is the live version of my project:
www.enviro-heroku.herokuapp.com

Javascript

Storyquiz is a site created to support online learning. The site will be targeted towards school teachers or online tutors and young children.
The site will be useful to them by proving a free reading quiz based on the books of their choice.
The user only needs to send the requested information through the contact page and will receive a quiz like the one shown on the about page.

Here is the live version of my project:
www.aria-vero-s.github.io/storyquiz

Django/bootstrap

PIZZAMAZE is a website for a restaurant with a booking system. The user can book one or more guests for a meal at a particular time and date. The data model "Booking" provides the features and business logic to manage, query and manipulate the customer bookings. I used Agile methodology to plan and design the website.

Here is the live version of my project:
www.pizzamaze.herokuapp.com
